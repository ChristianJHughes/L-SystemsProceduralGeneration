Monster Bash L-Systems Tool

By: Christian Hughes

Instructions: Enter input in the Rules, Axiom, and Iterations fields. The Axiom should be a single string without spaces. Rules should follow the pattern before->after. Multiple rules should be seperated by commas, and spaces are not allowed (Ex. A->B,B->AB). Press generate to draw the L-System/create an equivalent SVG snippet. Fill is off by default, and stroke is set to black by default.

Gramatical Components:

#(Color) - Use this to set the fill color. Color input should be a hexadecmial value with the hashtag ommited (ex. #(00008B)). Once set, the subsequent path is filled with color, until the fill color is changed. Specifying #(none) will turn off the fill.

|(Color) - Use this to set the stroke color. Color input should be a hexadecmial value with the hashtag ommited (ex. |(00008B)). Once set, the subsequent path is stroked with color, until the stroke color is changed. Specifying |(none) will turn off the fill.

+(Degrees) - Turn right by the specified degrees.

-(Degrees) - Turn left by the specified degrees.

[ - Saves the state (x value, y value, angle, fill color and state, stroke color and state).

] - Restores the last saved state.
